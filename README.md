# uglyfetch
If you want to run:
```bash
make && ./bin/uglyfetch
```

If you want to install:
```bash
make
sudo cp ./bin/uglyfetch /usr/local/bin/uglyfetch
sudo chmod +x /usr/local/bin/uglyfetch
```

Add to terminal startup, if you use bash:
```bash
echo "uglyfetch" >> ~/.bashrc
```
or if you use Zsh
```bash
echo "uglyfetch" >> ~/.zshrc
```
